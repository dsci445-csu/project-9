# project-9
Group 9 project for DSCI445 @ CSU

Kaggle Competition Page:
https://www.kaggle.com/competitions/child-mind-institute-problematic-internet-use/data

The packages we used for this project are knitr, readr, dplyr, caret, ggplot2, parsnip, mice, discrim, tidyr, lda, VIM, kableExtra, class, randomForest, ISLR, glmnet and tidyverse.
Please make sure these packages are installed and updated before you try to run any code.

- To replicate Lasso and Ridge regression, see 'Eddie_LinearRegression.Rmd'.
- To replicate Random Forest, see 'Kwan_RF.Rmd'.
- To replicate Linear Discriminant Analysis, see 'Nathan_LDA.Rmd'.

- To view the group presentation, see 'Group9_Presentation.pptx'.
- To view the project report, see 'Group9_Report.pdf'.
